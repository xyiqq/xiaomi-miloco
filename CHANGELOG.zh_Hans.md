# 更新日志

这个文件记录这个 fork 面向部署和维护的主要变动。

内容只保留和上游不同、并且适合公开发布的稳定改动。

## v0.0.5-fork - 2026-04-15

### 新增

- 增加了这个 fork 的公开部署说明。
- 增加了 Home Assistant 设备列表、分组、隐藏和恢复能力。
- 增加了 RTSP 摄像头新增、编辑、删除和持久化能力。
- 增加了设备管理页的搜索、按区域分组、区域批量选择和已隐藏设备恢复界面。
- 增加了摄像头列表中的 RTSP 内联管理入口。

### 变更

- README 和贡献说明已统一指向 `xyiqq/xiaomi-miloco`，不再默认引导用户去上游仓库部署、提 Issue 或参与讨论。
- 摄像头全屏显示改为保留 16:9 横向比例。
- RTSP 编辑时如果地址留空，会沿用原有流地址。

### 修复

- 修复了 Home Assistant 控制时 `service_data` 不能透传的问题。
- 修复了设备管理页面里 Home Assistant 区域信息显示不正确的问题。
- 修复了自托管部署中 RTSP 配置修改后不能持久化的问题。

### 说明

- 这份更新日志不包含站点专用的敏感信息或私有基础设施细节。
- 本地调试阶段用过、但没有保留为稳定源码的临时规则引擎实验，不属于这个版本的一部分。
